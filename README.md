# Agentic SDLC — Cheat Sheet

A single-page, interactive cheat sheet describing how agentic AI transforms every phase of the software development lifecycle.

Currently covers the **Discovery** phase in detail (Research, Synthesis, Workflow, Approval, QA agents). Placeholders for Design, Develop, Eval, Observe, Scale, Security, and Compliance phases.

## Live site

Hosted via GitHub Pages — see the repository Pages settings for the URL.

## Local preview

Open `index.html` in any modern browser. The page is a single self-contained HTML file with inline CSS/JS — no build step, no external dependencies beyond system fonts.
